# ShopEase

ShopEase is a React + TypeScript ecommerce frontend built with Vite.

## Tech Stack

- React
- TypeScript
- Vite
- React Router
- Context API
- Tailwind CSS
- Cypress

## Prerequisites

- Node.js 18+ (recommended Node.js 20+)
- npm 9+

## Setup and Run

1. Clone the repository:

```bash
git clone https://github.com/Kunal-Latta/ShopEase.git
```

2. Go to the project folder:

```bash
cd ShopEase
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
```

5. Open the app in your browser:

```text
http://localhost:5173
```

## Available Scripts

- Start dev server:

```bash
npm run dev
```

- Build production bundle:

```bash
npm run build
```

- Preview production build:

```bash
npm run preview
```

- Run Cypress tests (headless):

```bash
npm run test:e2e
```

- Open Cypress UI:

```bash
npm run test:e2e:open
```
